# Object Detection using YOLO

## 📌 About the Project

This project implements a real-time object detection system using the YOLO (You Only Look Once) algorithm to identify and classify objects in images and video streams. The system processes visual data and detects multiple objects simultaneously with bounding boxes and labels. It demonstrates the application of deep learning and computer vision techniques for real-world scenarios such as surveillance, automation, and intelligent systems.

---

## 🚀 Features

* Real-time object detection on images and video streams
* Detection and classification of multiple objects
* Bounding box visualization with labels
* Efficient and fast detection using YOLO algorithm
* Integration with OpenCV for image processing

---

## 🛠️ Technologies Used

* Python
* OpenCV
* YOLO (You Only Look Once)
* NumPy

---

## 📊 Dataset

The model uses pre-trained YOLO weights and can work on:

* Custom images
* Video files
* Live webcam feed

---

## ⚙️ Workflow

1. Load YOLO model and weights
2. Read input image/video/webcam stream
3. Preprocess input data
4. Perform object detection using YOLO
5. Draw bounding boxes and labels on detected objects
6. Display output in real-time

---

## ▶️ How to Run the Project

1. Clone the repository
   git clone https://github.com/samrudhi-shirode/Object-Detection-Project-Using-YOLO

2. Install dependencies
   pip install -r requirements.txt

3. Run the script
   python main.py   *(or your file name)*

## 🎯 Output

The system detects objects in real-time and displays them with bounding boxes and class labels on the screen.
